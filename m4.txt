*************** Характеристики ***************

Средняя выборочная
x- = sum[i = 1..k](xi * mi) / n
x- = ( 1.0259*5 + 1.0817*8 + 1.1375*10 + 1.1933*5 + 1.2491*2 ) / 30
x- = 1.12076

Исправленная выборочная дисперсия
S*^2 = sum[i = 1..k](mi * (xi - x-)^2) / (n - 1)
S*^2 = ( 5*(1.0259-1.12076)^2 + 8*(1.0817-1.12076)^2 + 10*(1.1375-1.12076)^2 + 5*(1.1933-1.12076)^2 + 2*(1.2491-1.12076)^2 ) / 29
S*^2 = 0.004112152137931034482758620690

Среднее квадратичное отклонение
S* = sqrt( S*^2 )
S* = sqrt( 0.004112152137931034482758620690 ) = 0.06412606441947794

Размах вариации
R = x_max - x_min
R = 1.277 - 0.998 = 0.279

Коэффициент вариации
V = (S* / x-) * 100%
V = ( 0.06412606441947794 / 1.12076 ) * 100% = 5.721658911763263%

S*3 = 0.0002636961328996594
