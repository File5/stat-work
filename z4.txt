*************** Характеристики ***************

Средняя выборочная
x- = sum[i = 1..k](xi * mi) / n
x- = ( 0.0444*5 + 0.0612*6 + 0.0780*10 + 0.0948*5 + 0.1116*4 ) / 30
x- = 0.07632

Исправленная выборочная дисперсия
S*^2 = sum[i = 1..k](mi * (xi - x-)^2) / (n - 1)
S*^2 = ( 5*(0.0444-0.07632)^2 + 6*(0.0612-0.07632)^2 + 10*(0.0780-0.07632)^2 + 5*(0.0948-0.07632)^2 + 4*(0.1116-0.07632)^2 ) / 29
S*^2 = 0.0004545037241379310344827586207

Среднее квадратичное отклонение
S* = sqrt( S*^2 )
S* = sqrt( 0.0004545037241379310344827586207 ) = 0.021319092948292408

Размах вариации
R = x_max - x_min
R = 0.12 - 0.036 = 0.084
